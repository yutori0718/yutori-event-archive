Googleスプレッドシートを編集すると、サイト側が公開CSVを読み込み、表示内容を更新できます。 JSONを直接編集しなくても、大会名・参加者・チーム構成・出場履歴を表で管理できます。

3. シートをCSVとして公開する

  1. Google Sheetsで ファイル → 共有 → ウェブに公開 を開きます。
  2. 公開したいシートを選びます。
  3. 形式を カンマ区切り値（.csv） にします。
  4. 公開URLをコピーします。
  5. 各シート分のURLを data/google-sheets.json に貼ります。
Google Sheets
→
CSV公開URL
サイトで表示

設定ファイル例

{
  "enabled": true,
  "sheets": {
    "wildcardParticipants": "https://docs.google.com/spreadsheets/d/e/xxxxx/pub?gid=0&single=true&output=csv",
    "wildcardMatchTeams": "https://docs.google.com/spreadsheets/d/e/xxxxx/pub?gid=123&single=true&output=csv"
  }
}
すべてのURLを一度に入れる必要はありません。使いたいシートだけURLを入れれば、その部分だけスプレッドシートが優先されます。

参加者一覧を更新する

5. 画像を使うとき

Google Sheetsには画像ファイルそのものを入れず、サイト内の画像パスを書きます。

画像を置く場所

images/
  wildcard-custom/
    waika-vol1/
      thumbnail.png
      team-list.png

シートに書くパス

/images/wildcard-custom/waika-vol1/thumbnail.png
/images/wildcard-custom/waika-vol1/team-list.png

反映されないときの確認

  • data/google-sheets.json の enabled が true になっているか
  • CSV公開URLの末尾が output=csv になっているか
  • シートの列名を消していないか
  • 画像パスが /images/... から始まっているか
  • GitHub Pagesの反映待ちやブラウザキャッシュではないか
Google Sheetsを非公開に戻すと、サイトから読み込めなくなります。 公開したくない情報はシートに書かないでください。
