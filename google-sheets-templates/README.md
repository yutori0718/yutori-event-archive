# Google Sheets Templates

このフォルダーのCSVをGoogleスプレッドシートへインポートして使います。

1つのスプレッドシート内に、下記と同じ名前のシートを作成してください。

- `siteContent`
- `apexEvents`
- `apexTeams`
- `wildcardEvents`
- `wildcardParticipants`
- `wildcardMatchTeams`
- `participationHistory`

各シートを「ウェブに公開」してCSV URLを取得し、`data/google-sheets.json` に貼り付けるとサイトに反映されます。
